# 🎂 Receita de Bolo de Cenoura com Cobertura de Chocolate

Página web desenvolvida como projeto prático de HTML5, CSS3 e JavaScript, exibindo a receita completa de um Bolo de Cenoura com Cobertura de Chocolate.

## 🔗 Acesse o projeto

> **Link do GitHub Pages:** `https://souzawendel818-art.github.io/BOLO-DE-CENORA-/`
>


---

## 📋 Sobre o Projeto

O objetivo deste projeto foi criar uma página web atrativa e bem estruturada aplicando conceitos fundamentais de desenvolvimento front-end:

- **HTML5 Semântico** — uso correto de `<header>`, `<main>`, `<section>`, `<footer>`, listas ordenadas e não ordenadas
- **CSS3 Estilizado** — paleta de cores harmoniosa, fontes do Google Fonts, Box Model (padding/margin) e espaçamento confortável
- **JavaScript** — botão interativo com alerta ao usuário
- **Responsividade** — layout legível tanto no desktop quanto no celular

## 🚀 Como Publicar no GitHub Pages

1. Crie um repositório no GitHub com o nome `receita-bolo`
2. Faça upload dos arquivos `index.html` e `style.css`
3. Acesse **Settings → Pages**
4. Em **Source**, selecione a branch `main` e a pasta `/ (root)`
5. Clique em **Save**
6. Aguarde alguns segundos e acesse o link gerado!
